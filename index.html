What is CoffeeScript

  1. A language that compiles into JavaScript.
  2. With Ruby-like syntax.
  3. Created by Jeremy Ashkenas.
  4. Announced on 2010.
  5. Filename extension: .coffee

Getting Started

  1. Install Node.js.
  2. Install CoffeeScript:
    npm install -g coffee-script
  3. To compile:
    coffee -c /path/to/my-coffeescript-file.coffee

"Hello, world!" Example

  1. CoffeeScript source: hello.coffee
    alert 'Hello, world!'
  2. Compile:
    coffee -c hello.coffee
  3. JavaScript output: hello.js
    alert('Hello, world!');

Why CoffeeScript

  1. It is JavaScript, with sugar.
  2. It probably compiles into better JavaScript comapred to handwritten.
