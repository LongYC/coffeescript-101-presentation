What is CoffeeScript

  1. A language that compiles into JavaScript.
  2. With Ruby-like syntax.
  3. Created by Jeremy Ashkenas.
  4. Announced on 2010.
  5. Filename extension: .coffee

Getting Started

  1. Install Node.js.
  2. Install CoffeeScript:
    npm install --global coffee-script
  3. To compile:
    coffee --compile /path/to/filename.coffee

"Hello, world!" Example

  1. CoffeeScript source: hello.coffee
    alert 'Hello, world!'
  2. Compile:
    coffee --compile hello.coffee
  3. JavaScript output: hello.js
    alert('Hello, world!');

Why CoffeeScript

  1. It is JavaScript, with sugar.
  2. It probably compiles into better JavaScript compared to handwritten.
  3. Instead of going through slides, let's do live coding to see why instead.
