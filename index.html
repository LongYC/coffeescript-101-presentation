What is CoffeeScript

  1. A language that compiles into JavaScript.
  2. With Ruby-like syntax.
  3. Created by Jeremy Ashkenas.
  4. Announced on 2010.
  5. Filename extension: .coffee

Getting Started

  1. Install Node.js.
  2. Install CoffeeScript:
    npm install --global coffee-script
  3. To compile:
    coffee --compile /path/to/filename.coffee

"Hello, world!" Example

  1. CoffeeScript source: hello.coffee
    alert 'Hello, world!'
  2. JavaScript output: hello.js
    (function() {
  alert('Hello, world!');
 
}).call(this);

Syntax

  1. Variable.
  2. Control flow.
  3. Function.
  4. Object.
  5. Everything is an expression.

Variable

  1. CoffeeScript syntax:
    score = 100
  2. Compile into JavaScript:
    var score;
 
score = 100;

Control Flow

  1. CoffeeScript syntax:
    grade = 'perfect' if score is 100
  2. Compile into JavaScript:
    var grade;
 
if (score === 100) {
  grade = 'perfect';
}

Function

  1. CoffeeScript syntax:
    add = (x, y) -> x + y
  2. Compile into JavaScript:
    var add;
 
add = function(x, y) {
  return x + y;
};

Object

  1. CoffeeScript syntax:
    number = value: 0
  2. Compile into JavaScript:
    var number;
 
number = {
  value: 0
};

Everything is an Expression

  1. Almost all statements can be used as expressions.
  2. Exceptions: break, continue, return
  3. Function returns are implicit.
  4. Encourage referential transparency.
