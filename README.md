# CoffeeScript 101

`0.3.0-SNAPSHOT`

A little talk on CoffeeScript.

## How to build

1. Keep calm.
2. Install [Node.js](http://nodejs.org/ "Node.js official website.").
3. $ `npm install`.

## How to use

1. Open `dist/index.html` with a web browser.
2. Use left and right arrow keys to navigate.
3. Have a nice day.
